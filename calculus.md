# 第一章：函数、图像和直线

## 1.1 函数

**函数**是把一个对象转化为另一个对象的规则。其实对象称为**输入**，来自称为**定义域**的集合。返回对象称为**输出**，来自称为**上域**的集合。
**上域：** 可能输出的集合。**值域：** 实际输出的集合。

判断函数相同：定义域、对应规则。

### 1.1.1 区间表示法
闭区间：[a, b]
&ensp;&ensp;开区间：(a, b)
&ensp;&ensp;半开闭：(a, b], [a, b)

### 1.1.2 定义域
（1）分数的分母不能是零
（2）不能取一个负数的偶次方根
（3）不能取一个负数或零的对数

$$f(x) = \frac{\log_{10}^{x + 8} \sqrt{26-2x}}{(x - 2)(x + 19)}$$
解得：
$$(-8, 2) \cup(2, 13]$$
也可以写成：
$$(-8, 13] \setminus  \{2\}$$
